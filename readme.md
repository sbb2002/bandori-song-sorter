뱅드림!(BanG Dream!) 13개 밴드의 수록곡 445곡을 **5단계**로 평가하고 나만의 랭킹을 만드는 웹앱입니다. 유튜브 영상을 페이지 안에서 바로 들으면서 곡을 평가할 수 있고, 결과는 브라우저에 자동 저장됩니다. PC와 모바일을 모두 지원합니다.

---

## 사용법

### 1. 밴드 선택

왼쪽 사이드바(모바일: 상단 바)에서 원하는 밴드 아이콘을 누릅니다. **ALL** 버튼을 누르면 전체 밴드의 곡을 한꺼번에 볼 수 있습니다.

### 2. 곡 재생

곡 행을 **짧게 누르면** 유튜브 영상이 재생됩니다.

- **PC**: 화면 중앙 오른쪽에 유튜브 플레이어가 임베드됩니다.
- **모바일**: 곡 리스트 위쪽 영역에서 재생됩니다.

♪ 표시가 있는 곡은 유튜브 링크가 등록되지 않아 재생을 지원하지 않습니다.

### 3. 랭크 매기기

곡 행을 **길게 누르면**(약 350ms) 랭크 팝업이 열립니다. PC에서는 **우클릭**으로도 팝업을 열 수 있습니다.

팝업에서 아래 5단계 중 하나를 선택하세요:

| 단계 | 이름 | 이모지 |
|:----:|------|:------:|
| 1 | 최애 | 💖 |
| 2 | 차애 | 💕 |
| 3 | 호 | 👍 |
| 4 | 중간 | 😐 |
| 5 | 불호 | 👎 |

이미 매긴 단계를 다시 누르면 평가가 **해제**됩니다.

#### 키보드 단축키 (팝업이 열려 있을 때)

| 키 | 동작 |
|----|------|
| `1` ~ `5` | 해당 단계 설정 (이미 설정된 경우 해제) |
| `Esc` | 팝업 닫기 (변경 없음) |

### 4. 필터로 걸러보기

곡 리스트 상단의 필터칩으로 특정 평가만 추려볼 수 있습니다.

**최애 / 차애 / 호 / 중간 / 불호 / 미평가** — 여러 개를 동시에 선택할 수 있습니다.

## 결과 저장

평가 결과는 **브라우저(localStorage)에 자동 저장**됩니다. 새로고침하거나 나중에 다시 방문해도 내용이 유지됩니다.

단, **다른 기기나 다른 브라우저에서는 저장이 공유되지 않습니다.** 시크릿(인코그니토) 모드에서는 탭을 닫으면 저장이 사라집니다.

초기화가 필요하면 하단 바의 **초기화** 버튼을 누르세요.

---

## 결과 공유

### 🔗 링크 복사

곡 리스트 상단의 **🔗 링크 복사** 버튼을 누르면, **현재 선택한 밴드와 티어 필터에 해당하는 곡들의 유튜브 URL**이 클립보드에 복사됩니다.

복사 형식은 DC인사이드 자동 렌더 형식(링크와 링크 사이 빈 줄)입니다:

```
https://www.youtube.com/watch?v=xxxxx

https://www.youtube.com/watch?v=yyyyy

https://www.youtube.com/watch?v=zzzzz
```

DC인사이드 게시글에 그대로 붙여 넣으면 유튜브 영상이 자동으로 삽입됩니다.

### ⬇ Download

하단 바의 **⬇ Download** 버튼을 누르면 통계 이미지를 PNG로 저장합니다.

이미지에 포함되는 내용:

- 전 밴드 개별 **히스토그램** (2열 그리드)
- **전체 밴드 히트맵**
- **베스트 밴드 사진** (최애→차애→호→불호 비율 기준으로 자동 선택)

> Download는 티어 필터와 관계없이 **전체 평가 결과**를 기반으로 이미지를 생성합니다. 특정 티어만 공유하고 싶다면 링크 복사를 사용하세요.

---

## 통계 보기

오른쪽 패널(모바일: 곡 리스트 아래)의 탭을 전환해 통계를 확인할 수 있습니다.

- **히스토그램 탭**: 현재 선택 밴드(또는 전체)의 5단계 분포를 막대 차트로 표시합니다.
- **히트맵 탭**: 모든 밴드 × 5티어 매트릭스로 밴드 간 선호도를 한눈에 비교합니다.

헤더 우측과 하단 바에서 **전체 평가 진행률**(평가한 곡 수 / 전체 곡 수)을 항상 확인할 수 있습니다.

---

## 모바일 지원

화면 너비 1023px 이하(태블릿·스마트폰)에서 세로 레이아웃으로 전환됩니다.

- 밴드 셀렉터: 상단 가로 스크롤 바 (스크롤 중에도 항상 보임)
- 유튜브 플레이어: 곡 리스트 위쪽에 배치
- 하단 바: 화면 하단에 고정(sticky)

## 개발자용 / 빌드

> 웹앱 사용만이 목적이라면 이 섹션은 건너뛰어도 됩니다. [라이브 링크](https://sbb2002.github.io/bandori-song-sorter/)에서 바로 사용할 수 있습니다.

### 의존성 설치

```bash
pip install pyyaml
```

### 빌드

`data/*.yaml`을 편집한 뒤 아래 명령어를 실행하면 `index.html`이 재생성됩니다.

```bash
python build.py
```

### 테스트

```bash
npm test   # node --test — core.js 순수 함수 단위 테스트 (16개)
```
