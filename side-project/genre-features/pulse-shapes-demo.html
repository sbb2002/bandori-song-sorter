실제 크기(≈46px)로 여러 곡이 섞여 박마다 발산 — 작게 겹쳐도 모양이 구분되는지 확인.

채널 배정: 곡 피처를 코퍼스 z-score로 표준화 → acoustic=z(harmonic_ratio) · bright=mean z(centroid,rolloff,zcr,flatness) · shimmer=z(flux) 중 최댓값. 최댓값−2등 < 0.4면 neutral. 색은 밴드 정체성(현행 유지)이라 모양만 채널로 바뀐다. 프로토타입 데모 — 실제 반영 전 UX 판단용.
